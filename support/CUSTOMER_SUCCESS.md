## Response SLAs

| Priority | Definition | First response | Resolution |
|----------|-----------|----------------|------------|
| Urgent | Service down, data loss risk | < 1 hour | < 4 hours |
| High | Core feature broken | < 4 hours | < 24 hours |
| Normal | General question, non-critical bug | < 24 hours | < 3 days |
| Low | Feature request, feedback | < 3 days | Logged to roadmap |

## Ticket Response Templates

### Greeting + confirming receipt
```
Hi [Name],

Thanks for reaching out! I've got your message and I'm looking into it now.

[Your name]
[COMPANY] Support
```

### Bug confirmed — working on it
```
Hi [Name],

Thanks for reporting this — you've found a real bug. I've logged it and our team is working on a fix.

I'll follow up as soon as it's resolved. Apologies for the inconvenience.

[Your name]
```

### Feature doesn't exist yet
```
Hi [Name],

Great question — this isn't something we currently support, but I've logged your request.

In the meantime, [workaround if any].

We'll let you know when this ships. Thanks for the feedback — it directly shapes what we build.

[Your name]
```

### Billing issue — refund approved
```
Hi [Name],

I've reviewed your account and I'm happy to issue a refund of $[X]. You should see it back on your card within 5–10 business days.

Let me know if there's anything else I can help with.

[Your name]
```

### Closing a resolved ticket
```
Hi [Name],

Just following up — is everything working well now?

If you run into anything else, don't hesitate to reach out.

[Your name]
```

## Proactive Outreach

Don't wait for users to come to you. Reach out proactively:

### New paid customer (day 1)
Personal email from founder:
```
Subject: Welcome to [PRODUCT] — anything I can help with?

Hi [Name],

Thanks for upgrading — really glad to have you on board.

If you run into anything or have questions, reply directly to this email. I read every response.

[Founder name]
```

### User stuck in activation (day 3, no Aha Moment)
```
Subject: Having trouble getting started?

Hi [Name],

I noticed you signed up a few days ago but haven't [completed X] yet.

[One specific tip to get them unstuck]

Happy to help — just reply here.

[Your name]
```

### Win-back (after cancellation, day 14)
```
Subject: What could we have done differently?

Hi [Name],

I saw you cancelled your [PRODUCT] subscription recently.

Would you mind sharing what led to that decision? Even one sentence helps us improve.

If anything has changed or you want to give it another try, I'm happy to offer [X].

[Your name]
```

## Cancellation Save Flow

When a user initiates cancellation:

1. **Show exit survey** — capture reason (required)
2. **Based on reason, show targeted offer:**

| Reason | Save offer |
|--------|-----------|
| Too expensive | Offer [X]% discount or plan downgrade |
| Missing feature | Show roadmap item or workaround |
| Not using it | Offer to help or pause subscription |
| Switching to competitor | Show comparison, offer extended trial |
| Business/project ended | Acknowledge, offer to pause |

3. **If they still cancel:** thank them, send farewell email, start win-back sequence (see `marketing/playbooks/EMAIL.md`)
