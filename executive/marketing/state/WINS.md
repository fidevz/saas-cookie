# WINS

> Validated learnings from things that worked.
> The agent updates this file when something significantly outperforms expectations.
> Reference this before creating new content or campaigns — don't reinvent what already works.

---

## Format
Each win includes:
- **What:** description of the content/ad/campaign
- **Where:** platform and context
- **Result:** the metric that made it a win
- **Why it worked:** hypothesis for the reason
- **How to replicate:** the pattern to reuse
