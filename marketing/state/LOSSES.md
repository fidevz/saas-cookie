# LOSSES

> Documented failures. The agent updates this when something is killed or significantly underperforms.
> Read this before launching anything similar — don't repeat mistakes.

---

## Format
Each loss includes:
- **What:** description of what failed
- **Where:** platform and context
- **Result:** the metric that made it a failure
- **Why it failed:** honest hypothesis
- **What NOT to do:** the pattern to avoid
